== Description ==

Provides an admin interface for manually creating donation transactions in Give.

More information at [givewp.com](http://givewp.com/addons/manual-donations).

== Installation ==

1. Activate the plugin
2. Go to Downloads > Settings > PDF Receipts and configure the options
3. Customers can now download receipts for any donation on the Donation History page
